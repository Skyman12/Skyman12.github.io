Project Description

This application allows a user to build winning League of Legends Teams.


Documentation
GitHub Server Repository
GitHub Client Repository

Project Features

  • Can view individual summoners stats for every ranked game.
  • Uses a "secret algorithem" to calculate Mid, Top, Jungle, Support and ADC scores for every summoner.
  • Builds a lineup based on who plays which role best and team composition.
  • Can assign a specific player a role or champion and build a team around them.

Languages and Frameworks Used

  • Dart
  • pHp
  • CURL
  • LOL API
  • UWAMP
  • Amazon Webservices

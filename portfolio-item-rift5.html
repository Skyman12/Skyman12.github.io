Project Description

Space flight simluation designed to be used with the Oculus Rift. Won "Best Project" award out of 40 teams.

Project Features

  • Time trial mode allows a user to race for the fastest time through a course.
  • Multiplayer race mode for up to 4 person races.
  • Voxel system for ship building.
  • Integration with the Oculus Rift.

Languages and Frameworks Used

  • Unreal Engine
  • C++
  • pHp
  • Oculus Rift

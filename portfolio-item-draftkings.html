Project Description

This application allows a user to view and build DraftKings lineups.


Documentation
GitHub Repository

Project Features

  • Can view individual players projections for the upcoming fantasy week.
  • Can find players who have a high projection / salary ratio quickly. These players are good sleeper picks.
  • Can build a lineup with specific players and have the program automatically finish the lineup with the best possible team.
  • Can generate the "ideal lineup", based on projections.

Languages and Frameworks Used

  • Java
  • JavaSwing
  • FantasyFootballNerds projection API
  • DraftKings salary API

Project Description

A 4 way stoplight with turn signals, pedestrian input and police control.


Project Files on Dropbox

Project Features

  • Automatic mode shuffles through 4 60 second preset light configurations when no input is being recieved.
  • Input mode takes in weight input from cars and button input from pedestrians.
    • Allows the controller to determine the optimal way to allow all cars / pedestrians to cross.
    • Once everyone has been given an opportunity to cross, the input mode takes in new sensor input and repeats the process.
  • Police mode will automatically shut down all lights in 5 seconds and turn the lights green in the approaching police cars direction.
  • Timer counts down the time left in the cycle. Displayed in the upper right of the FPGA board. A 'P' is displayed when the cycle is in Police Mode.

Languages and Frameworks Used

  • Verilog
  • Arduino FPGA Board
  • Altera Quartus II
  • Hades
